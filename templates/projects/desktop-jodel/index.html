Desktop Jodel Client

A inofficial, Qt-based desktop client for the Jodel chat protocol.

Source code on Github.com

Features
  • Basic functions of the Jodel chat protocol (voting, post colors, images)
  • Display of posts' (coarse) locations
  • Manual setting of location
  • Manual selection of post color
  • Nice emoji images
Used Technologies
  • Qt 5
  • Emoji images from emojione
