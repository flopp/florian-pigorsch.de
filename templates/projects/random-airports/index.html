Random Airports

Random Airports is a web app that displays satellite views of random airports.

{{ image("airports-screenshot.jpg", "Random Airports") }} {{ buttons2("https://airports.flopp.net/", "Jump to the \"Random Airports\" website", "https://github.com/flopp/airports_map", "Source code on Github.com") }}

Features

  • 'Auto play' mode
  • Search function for ICAO codes, IATA codes, airport names and location names
  • Automatic fitting of view
  • Smooth fading to next airport

Used Technologies

  • Google Maps Javascript API for mapping and as a provider for satellite imagery
  • The frontend site queries airport data from a PHP/sqlite3 backend via AJAX calls
  • Airport data is imported from OurAirports.com
