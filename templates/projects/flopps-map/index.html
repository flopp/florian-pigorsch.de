Flopp's Map

Flopp's Map is a single page web app that allows for simple geodesic operations, like determining the coordinates of a specific location, measuring the distance between several locations or performing a waypoint projection.

The map attracts several hundred users per day, among them a lot of geocachers who use the map to perform geocaching related operations, for planning geocaching tours, and as a simple means to explore the world before actually going outside.

Jump to the Flopps's Map website Source code on Github.com

Features

  • Draggeable markers for determining coordinates
  • Geodesic operations: waypoint projection, distancce/bearing measurement
  • Different mapping styles
  • Searching by coordinates and location names
  • Multi language interface
  • Generation of permalinks to share the current map view with others

Used Technologies

  • Google Maps Javascript API for mapping
  • Opencaching API (OKAPI) for accessing the data bases of Opencaching sites
  • Bootstrap for the user interface
  • jQuery and various plugins for Javascript stuff
