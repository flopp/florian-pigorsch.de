QtAsciimage Library + Editor

QtAsciimage is a C++/Qt library for rendering images specified in the Asciimage format. Along with the library comes a graphical editor for that kind of images - QtAsciimageEditor.

Source code on Github.com

Features

  • Library: extended Asciimage format for specifying rendering styles
  • Library: direct rendering to QImage
  • Library: QIconEngine support for providing Qt applications with resolution independant icons
  • Editor: grid template
  • Editor: instant preview
  • Editor: preview scaling

Used Technologies

  • Qt, the cross platform UI framework
  • @cparnot's Asciimage specification
