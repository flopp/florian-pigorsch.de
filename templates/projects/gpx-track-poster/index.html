GPX Track Poster

Generator for visually appealing posters from your GPS tracks.

Source code on Github.com

Features
  • 'grid', 'calendar' and 'heatmap' modes
  • Track statistics at the poster's footer (total, average, max track length, etc.)
  • Highlighting of 'special' tracks (i.e. events, competitions)
  • Fully customizable via command line options
  • Caching of parsed track data for faster subsequent executions of the generator
Used Technologies
  • gpxpy for parsing GPX files
  • svgwrite for generating SVG files
