Safari Map

The Safari Map displays Safari caches listed on the geocaching site Opencaching.de.

In contrast to physical geocaches where the goal is to find a physical geocaching container in the woods, Safari caches are virtual geocaches that require the completion of a specified task (e.g. find a specified geological feature, or a specified type of building). These tasks are not bound to a single location but can be fulfilled at multiple locations worldwide.
The Safari Map displays the Safari caches along with the locations where the geocachers have completed the corresponding tasks.

Jump to the Safari Map website Source code on Github.com

Features

  • Display of Safari Caches from Opencaching.de
  • Filtering by name, owner, tags
  • Placement of log markers at coordinates extracted from the individual log texts
  • Display of cache logs including log images
  • RSS feed containing recently added safari caches

Used Technologies

  • Opencaching API (OKAPI) for accessing the Opencaching.de data base
  • Leaflet for mapping
  • Bootstrap for the user interface
  • jQuery and various plugins for Javascript stuff
