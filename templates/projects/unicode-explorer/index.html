Unicode Explorer

Unicode Explorer is an experimental webapp that let's you explorer Unicode code blocks and individual Unicode characters.

Features

  • Responsive layout
  • Display of individual Unicode characters including related and look-alike characters
  • Display of Unicode code blocks

Used Technologies

  • Flask microframework
  • Milligram CSS framework
  • Unicode data from Unicode.org
